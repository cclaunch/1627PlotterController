Attaching a 1627 Plotter to an IBM 1130 system required the installation of an IBM double width SLT card 5806223 into
gate A-C1 in slots M4/M5. The 1627 was an IBM badged version of the Calcomp 565 or 563 plotters.

This project provides a functional equivalent to that IBM SLT card that can be plugged into the IBM 1130 to add 
the controller logic so that software can command plotter movements and see its status in the same way as
the IBM card.
